В репозитории храняться файлы исходного кода, а также
все необходимые файлы для запуска .exe.

Также приложено изображение результата работы алгоритма с файлом test.vtk (test_result)
и полный результат работы с основными данными (main_result)

Запуск программы:
1) изменить директории в настроечном файле: ray_tracing3d_start_settings_file.txt
(указать путь файла test.vtk и out)
2) вызвать из консоли ray_tracing3d.exe, указав в качестве входного параметра
полный путь до настроечного файла ray_tracing3d_start_settings_file.txt
